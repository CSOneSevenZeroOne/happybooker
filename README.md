### 欢乐书客

* 文件夹 ：'./src'
* 服务端: './server'
* 数据库/名: happybooker.sql&happybooker

    此项目由React脚手架构建

    ```
    $ cnpm install -g create-react-app
    $ create-react-app happybooker
    ```

输入 `npm start` 命令运行 react-scripts start,打开 http://localhost:3000 主页面
